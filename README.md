# SabiFit

> Nigerian fashion designers' best friend. Digitize measurements, manage clients, track orders.

## Features

- **Client Management** — Add, search, and archive clients with full measurement profiles
- **Measurement Capture** — 16 Nigerian garment types with built-in measurement guides
- **Order Tracking** — Track orders from start to delivery with status updates
- **Invoices** — Auto-generate invoices with Paystack payment links
- **Cloth Calculator** — Built-in yardage calculator for fabric estimates
- **Self-Service Links** — Send measurement links to diaspora clients

## Quick Start

### 1. Create Account

Go to [sabifit.app](https://sabifit.app) and create your shop.

### 2. Add Clients

Add your first client with their measurements.

### 3. Create Orders

Create orders and track progress.
