# SabiFit

African fabric & clothing size calculator and fitness tracking app. Built with Next.js 15 (App Router).

## Features
- African fabric (Ankara, lace, etc.) yardage calculator
- Clothing size conversion charts (US, UK, EU, African sizing)
- Body measurements tracker
- Nigerian fabric types database

## Stack
- Next.js 15 (App Router, TypeScript)
- React 19
- Tailwind CSS
- Vercel deployment
